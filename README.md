プロジェクトは自分で作る必要がある。

TMDb APIキーなどの知識は必要。

どのディレクトリにファイルを生成する必要があるは考える必要がある。

Xcodeの操作方法は知っておく必要がある。（プロジェクト作成、ファイル作成など）

ビルドなどの用語は知ってる必要がある（質問できないから）
